# TaskFlow 디자인 가이드

## 1. 전체적인 무드 (Overall Mood)

TaskFlow는 **신뢰할 수 있는 전문적인 도구**로서의 정체성을 가집니다. 1인 개발자가 복잡한 프로젝트를 효율적으로 관리할 수 있도록 돕는 AI 기반 태스크 매니저로, 차분하고 정돈된 인터페이스를 통해 높은 정보 밀도를 제공하면서도 개발자 친화적인 경험을 선사합니다. 

전체적인 디자인 무드는 **미니멀하면서도 기능적**이며, 불필요한 장식보다는 **명확한 정보 전달과 효율적인 워크플로우**에 집중합니다. 다크 모드를 기본으로 하여 장시간 작업하는 개발자들의 눈의 피로를 줄이고, 코딩 환경과의 일관성을 유지합니다.

## 2. 참조 서비스 (Reference Service)

- **서비스명**: Linear
- **제품 설명**: 현대적인 이슈 트래킹 및 프로젝트 관리 도구
- **디자인 무드**: 미니멀하고 기능적인 인터페이스, 높은 정보 밀도, 개발자 중심의 UX
- **주요 색상**: #5E6AD2 (보라빛 파랑)
- **보조 색상**: #F6F8FA (연한 회색)

Linear의 깔끔하고 효율적인 인터페이스, 키보드 단축키 중심의 상호작용, 그리고 개발자가 선호하는 다크 모드 우선 접근 방식을 참조합니다.

## 3. 색상 & 그라데이션 (Color & Gradient)

### 주요 색상 팔레트
- **Primary Color**: #2563EB (Azure Blue) - 주요 CTA, 선택 상태, 링크
- **Primary Hover**: #1E4FD1 - 버튼 호버 상태
- **Secondary Background**: #0F172A (Gunmetal Slate) - 다크 모드 배경, 사이드바
- **Success Accent**: #22C55E (Emerald) - 완료 및 성공 상태 뱃지
- **Warning Accent**: #FACC15 (Sun Yellow) - 긴급 태스크, 경고 알림

### 중성 색상
- **Neutral-100**: #F8FAFC - 라이트 모드 주요 배경
- **Neutral-200**: #F1F5F9 - 카드 및 섹션 배경
- **Neutral-600**: #475569 - 본문 텍스트
- **Neutral-900**: #0F172A - 타이틀 및 헤드라인

### 색상 무드 및 사용법
- **무드**: 차가운 톤 (Cool), 낮은 채도 (Low Saturation)
- **색상 사용 우선순위**:
  1. Primary Blue: 가장 중요한 액션 요소 (CTA 버튼, 활성 상태)
  2. Success Green: 완료된 태스크, 성공 메시지
  3. Warning Yellow: 주의가 필요한 요소, 긴급 태스크
  4. Neutral Gray: 일반 텍스트, 배경, 구분선

## 7. UX 가이드 (UX Guide)

### 타겟 사용자 맞춤 전략
TaskFlow는 **전문 개발자**를 대상으로 하므로 다음 UX 원칙을 따릅니다:

1. **키보드 우선 인터랙션**
   - 모든 주요 기능은 키보드 단축키로 접근 가능
   - 명령 팔레트 (`/` 키)를 통한 빠른 네비게이션

2. **정보 밀도 최적화**
   - 한 화면에 최대한 많은 유용한 정보 표시
   - 불필요한 공백이나 장식 요소 최소화

3. **실시간 피드백**
   - SSE를 통한 즉시적인 상태 업데이트
   - 낙관적 UI로 반응성 향상

4. **컨텍스트 유지**
   - 작업 중인 태스크와 관련 정보를 항상 시각적으로 연결
   - 브레드크럼과 상태 표시를 통한 현재 위치 명확화

### 입력 필드

- **패딩**: 12px 16px
- **둥근 모서리**: 6px
- **플레이스홀더**: #94A3B8

### 카드
- **배경**: 라이트 모드 #F1F5F9, 다크 모드 #1E293B
- **그림자**: shadow-sm
- **둥근 모서리**: 8px
- **패딩**: 20px

### 칸반 컬럼
- **너비**: 280px
- **컬럼 헤더**: 14px, 600 weight
- **아이템 간격**: 8px
- **드래그 상태**: 미묘한 그림자 + 액센트 테두리

### 태스크 아이템
- **헤더**: 타이틀 14px 500 weight, 상태 뱃지 우측 정렬
- **드래그 시**: 미묘한 elevation + 액센트 테두리
- **호버**: 배경색 약간 진하게

### 사이드바
- **너비**: 240px
- **배경**: Slate 계열
- **구분선**: #374151, 1px

### 차트
- **기본 색상 팔레트**: 
  - Primary 60% (#2563EB)
  - Accent 20% (#22C55E)
  - Neutral 20% (#94A3B8)

### 토스트 알림
- **위치**: 우상단
- **자동 해제**: 3초
- **색상**: 성공(#22C55E), 경고(#FACC15), 오류(#EF4444)

### 상호작용 가이드라인
- **키보드 단축키**:
  - G K: 칸반으로 이동
  - G D: 의존성 그래프로 이동  
  - G T: 타임라인으로 이동
  - /: 명령 팔레트 열기
- **드래그 앤 드롭**: dnd-kit 사용하여 칸반에서 구현
- **실시간 업데이트**: SSE를 통한 낙관적 UI, 300ms 미만 로딩 스피너

### 접근성 가이드라인
- **색상 대비**: WCAG AA 기준 준수
- **포커스 아웃라인**: Primary 색상, 2px 두께
- **ARIA 레이블**: 버튼 및 상호작용 아이콘에 적용
